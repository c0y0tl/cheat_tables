# BloodRayne Cheat Table
* Debug Mode [0 - off / 1 - on]
* Disable AI [0 - off / 1 - on]
* God Mode [0 - off / 1 - on]
* Fly Mode  [0 - off / 1 - on]
    * Q/A - Move up/down
    * Arrows - Move sideways
    * Numpad 1/3 - Rotate camera left/right
    * Numpad 7/9 - Rotate model
    * Numpad 5 - Reset model rotation
    * Shift - Speed up movement
